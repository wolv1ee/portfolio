# Portfolio — Saad Mahmud

Personal portfolio built with Astro, deployed on Vercel.

🌐 [portfolio-beryl-beta-40.vercel.app](https://portfolio-beryl-beta-40.vercel.app)

## Stack
- Astro
- Lenis (smooth scroll)
- GSAP (animations)
- Vercel (hosting)

## Run locally
```bash
npm install
npm run dev
```
